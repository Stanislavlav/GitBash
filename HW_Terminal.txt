Linux terminal (GitBash) commands

1) Посмотреть где я -- pwd

2) Создать папку -- mkdir QA

3) Зайти в папку -- cd QA

4) Создать 3 папки -- mkdir qa1 qa2 qa3

5) Зайти в любую папку -- cd qa1

6) Создать 5 файлов (3 txt, 2 json) -- touch file1.txt file2.txt file3.txt file4.json file5.json

7) Создать 3 папки -- mkdir papka1 papka2 papka3

8) Вывести список содержимого папки -- ls -l(отображает без скрытых файлов списком) или ls -la(отображает скрытые файлы) или ls(отображает без скрытых файлов и в одну строку)

9) + Открыть любой txt файл -- cat >> file1.txt

10) + написать туда что-нибудь, любой текст. --
number1
number2
number3

11) + сохранить и выйти. -- ctrl+C

12) Выйти из папки на уровень выше -- cd ..

—
13) переместить любые 2 файла, которые вы создали, в любую другую папку. -- mv qa1/file1.txt qa1/file4.json qa2

14) скопировать любые 2 файла, которые вы создали, в любую другую папку. -- cp qa1/file2.txt qa1/file5.json qa3

15) Найти файл по имени -- find . -name "file5.json"

16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. 
 -- cd qa2
 -- tail -f file1.txt
 После этого открываем в блокноте файл file1.txt и пишем новую строку number4, нажимаем сохранить. В терминале отображается новая строка в реальном времени.

17) вывести несколько первых строк из текстового файла -- head -n5 filename

18) вывести несколько последних строк из текстового файла -- tail -n5 filename

19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
-- less filename

20) вывести дату и время
-- date
